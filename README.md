# SitStandTracker

A tiny native macOS app for tracking time spent sitting and standing.

## Features

- One-click switching between `Sitting` and `Standing`
- Live timer for the current posture
- Editable sit/stand work-cycle goals
- Daily totals, goal status, and recent analytics
- Day-grouped session history
- Menu bar controls and transition alerts
- Screen-lock suspend/resume behavior
- Local persistence via `UserDefaults`

## Run

From the repo root:

```bash
swift run SitStandTracker
```

## Open In Xcode

```bash
open SitStandTracker.xcodeproj
```

Then run the `SitStandTracker` scheme.
